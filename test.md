# Таблица переходов и состояний: Страница авторизации
## https://applicant.21-school.ru/auth

---

## Описание состояний системы

| ID | Состояние | Описание |
| :--- | :--- | :--- |
| S0 | `Ожидание авторизации` | Начальное состояние приложения перед началом взаимодействия пользователя. Отображается главная страница с формой входа и ссылками на регистрацию и восстановление пароля. |
| S1 | `Форма входа` | Интерфейсная страница, содержащая поля для ввода логина и пароля, кнопку "Войти" и ссылку "Забыли пароль?". |
| S2 | `Форма регистрации` | Интерфейсная страница, содержащая поля для ввода данных нового пользователя (например, имя, email, пароль), кнопку "Зарегистрироваться" и ссылку на страницу входа. |
| S3 | `Профиль пользователя` | Главная страница приложения, отображаемая после успешной аутентификации. Содержит информацию о пользователе и кнопку выхода из системы. |
| S4 | `Успешная регистрация` | Страница-уведомление, отображаемая после успешного создания нового аккаунта. Сообщает пользователю о том, что регистрация завершена, и предоставляет ссылку для входа. |
| S5 | `Ошибка входа` | Уведомление об ошибке, отображаемое на форме входа при вводе неверных учетных данных. Содержит сообщение "Неверный логин или пароль". |
| S6 | `Форма сброса пароля` | Интерфейсная страница для восстановления доступа, содержащая поле для ввода электронной почты и кнопку "Отправить". |
| S7 | `Письмо отправлено` | Страница-уведомление, отображаемая после отправки запроса на сброс пароля. Сообщает пользователю, что на указанный адрес электронной почты было отправлено письмо с инструкциями. |
| S8 | `Новый пароль` | Интерфейсная страница, открывающаяся по ссылке из письма для сброса пароля. Содержит поля для ввода нового пароля и подтверждения, а также кнопку "Сохранить". |
| S9 | `Пароль изменен` | Страница-уведомление, отображаемая после успешного изменения пароля. Сообщает пользователю, что его пароль был успешно обновлен, и предоставляет ссылку для входа в систему. |
